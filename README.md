# TG bot wrapper

Этот репозиторий содержит в себе обертки и классы, которые на прямую работают с tg api, но при этим упрощают работу с загрузкой команд, запуском бота, сзаимодейсвтием с сообщениями и т.п.

# Структура

## bot_father_info

[Папка](./bot_father_info/)

Содержит описание команд, описание бота, о чем бот, логотип и задний фон

## bot

[Папка](./bot/)

Содержит основной функционал для запуска и взаимодествия с ботом

## message

[Папка](./message/)

Класс `message` позволяет хранить сообщения в определенной папке, загружать и её в память для дальнейшего использования

**Note:**
Для определения своего сообщения необходимо отнаследоваться от класса `nMessageType`. Сам класс `nMessageType` представляет собой `Enum` класс. Орпределять названия команд необходимо в производном, как в примере


```py
class MyText(nMessageType):
    ERROR : str = "error" # `error` - название файла в этом случаи
    WELCOME : str = "welcome"
```

## parameters

[Папка](./parameters/)

Содержит `PARAMETERS`, как `dict` переменных окружения.

Неоходимо инициализация параметров через `init_parameters` или через декоратор `dec_init_parameters`
